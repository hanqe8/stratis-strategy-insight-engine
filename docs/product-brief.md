# Product Brief

## Thesis

Stratis - Strategy & Insight Engine is a browser-based executive strategy workbench that converts evidence, assumptions, financial inputs, market signals, and strategic options into decision-ready briefings with citations, scoring logic, sensitivity checks, and recommendation trails.

## Primary User

Strategy or product professionals preparing executive-ready recommendations.

## MVP Boundary

The MVP is static and client-side only. It supports local project creation, evidence capture, assumption tracking, scoring, sensitivity analysis, pre-mortems, CSV charting, deterministic brief generation, Markdown export, and JSON backup.

Excluded from v0.1: login, server database, live scraping, confidential document ingestion, paid API dependency, complex PDF parsing, and automated financial data feeds.
