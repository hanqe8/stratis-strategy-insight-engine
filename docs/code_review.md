# Code Review Checklist

- Static hosting compatibility is preserved.
- No secrets, private documents, or proprietary data are committed.
- TypeScript strict mode passes.
- Scoring, sensitivity, and brief generation logic have tests.
- Tables and charts remain usable on mobile widths.
- Recommendation logic remains visible and auditable.
- Accessibility basics are present: labels, focus states, semantic headings, and keyboard-operable controls.
